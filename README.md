# Sistema de Cifrado Clásico: César & Atbash

Aplicación web desarrollada en Angular para la demostración de 2 métodos de encriptación: César & Atbash. 

## Sitio Web de la Aplicación
La versión estable de este sistema está publicada y accesible en: 
- `https://cifrado-b11ab.web.app/`

## Instrucciones de Ejecución Local
Si desea compilar y ejecutar el código fuente en un entorno local:
1. Asegúrese de tener Node.js y Angular CLI instalados.
2. Clone este repositorio.
3. Ejecute `npm install` para descargar las dependencias.
4. Ejecute `ng serve` para levantar el servidor de desarrollo.
5. Navegue a `http://localhost:4200/`.
